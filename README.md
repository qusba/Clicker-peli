# ot-hajoitustyö
Ohjelman perusidea on clicker-tyypin peli, jossa pisteitä kerätään hiiren painalluksilla. Pisteillä voi ostaa kaupasta päivityksiä,
jotka nopeuttavat pisteiden keräämistä.
